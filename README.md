# Personal Portfolio Website

A simple personal portfolio website built using  HTML5 to showcase personal information, skills, and contact details.

## Features

- About Me section
- Skills section
- Contact form with validation
- Responsive design
- Internal navigation links
- Profile image and social links
- Semantic HTML5 structure

## Technologies Used

- HTML5

## Project Structure

```text
Portfolio/
├── index.html
├── profile.jpg
└── README.md
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/personal-portfolio.git
```

2. Open `index.html` in your browser.
